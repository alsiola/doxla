# Changelog

All notable changes to this project will be documented in this file.

## [2.3.0] - 2026-02-01

### Added
- GraphQL API alpha release
- Widget templates marketplace
- Dark mode support in dashboard

### Fixed
- Memory leak in widget engine when handling large payloads
- Dashboard failing to load on Safari 18

## [2.2.0] - 2026-01-15

### Added
- Batch widget deployment via CLI
- Webhook notifications for widget lifecycle events

### Changed
- Upgraded to Node.js 22 runtime
- Improved cold start times by 40%

## [2.1.0] - 2025-12-01

### Added
- Multi-region deployment support
- Custom domain mapping for widgets
- Real-time collaboration in dashboard

### Deprecated
- Legacy REST API v1 endpoints (will be removed in 3.0)
