# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Doxla, please report it by opening a GitHub issue at https://github.com/alsiola/doxla/issues.

We ask that you practice responsible disclosure:

- Provide a clear description of the vulnerability and steps to reproduce it.
- Allow reasonable time for a fix before publicly disclosing details.
- Do not exploit the vulnerability beyond what is necessary to demonstrate it.
